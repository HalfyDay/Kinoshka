# Киношка

`Киношка` — удобное Android-приложение для поиска фильмов и сериалов, просмотра подробной информации и быстрого перехода к плееру.

## Почему стоит попробовать
- Быстрый поиск по каталогу
- Красивые карточки с рейтингами и деталями
- Личная библиотека: статусы, заметки, прогресс по сериалам
- История просмотра
- Профиль с кастомной аватаркой и статистикой активности
- Встроенные настройки внешнего вида (включая AMOLED-тему)

## Что внутри
- Вкладки библиотеки: `История`, `Смотрю`, `В планах`, `Просмотрено`, `Пересматриваю`, `Отложено`, `Брошено`
- Удобное окно редактирования прогресса
- Импорт/экспорт локальной библиотеки (резервная копия)
- Встроенное открытие страницы плеера

## Скачать APK
1. Откройте раздел **Releases** в этом репозитории.
2. Скачайте последний `app-release.apk`.
3. Установите файл на Android-устройство.

## Первый запуск
1. Откройте приложение.
2. Найдите фильм или сериал.
3. Добавьте тайтл в библиотеку и ведите личный прогресс.

## Требования
- Android 8.0+ (API 26 и выше)
- Подключение к интернету

## Важно
Приложение использует данные Kinopoisk Unofficial API. Доступность некоторых функций зависит от внешних сервисов и сети.
